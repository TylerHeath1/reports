📋 Required Jira fields per Epic (so this page renders fully)

Each Epic owner please populate these fields directly in Jira. The roadmap reads them on each refresh.

  • Start date + Due date — controls the bar's start and end positions on the timeline
  • Project Health — On Track / At Risk / Off Track
  • Product Execution Update — short narrative shown when someone clicks the Epic bar
  • Funding Mechanism — funder/grant tag (e.g., PJM, HSF, General No Donor)
  • Assignee — Epic owner
  • Parent — must be linked to an Initiative (the user-story-level row above it)
